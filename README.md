# Blue Noise Dither

A free, fast, client-side tool for applying blue-noise dithering and halftone effects to images, with a real-time preview. All processing happens in the browser — images never leave your device.

Live at **[blue-noise.blode.co](https://blue-noise.blode.co)**.

## Stack

- Next.js 16 (App Router), React 19, React Compiler
- Tailwind CSS v4
- Radix UI primitives, blode-icons-react
- Biome (lint + format)

## Development

```bash
npm install
npm run dev          # start dev server
npm run build        # production build
npm run lint         # Biome check
npm run check-types  # TypeScript
```
